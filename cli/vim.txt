# replacing
:%s/old/new/g[c]               - Replace all matches in file [with confirmation].
:s/old/new                     - Replace first match on line.
:s/old/new/g                   - Replace all matches on line.
:#,#s/old/new/g                - Replace matches between two lines with the number #
Good Cheatsheet: https://vim.rtorr.com

# register
"_d        # delete something and copy to clipboard

# outside actions
:!command                      - Execute external command.
:w FILENAME                    - Save file with new name.
v motion :w FILENAME           - Save visually selected lines to file.
:r FILENAME                    - Insert file contents below cursor.
:r !command                    - Insert output of external command below cursor.

# Intellij Ideavim
:actionlist          # see a list with all actions
:vs                  # vertical split
:sp                  # split down
ctr-w h              # go left window
ctr-w j              # go lower window
ctr-w k              # go upper window
ctr-w l              # go right window
